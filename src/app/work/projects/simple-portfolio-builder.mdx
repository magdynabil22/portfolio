## Overview

Developed a full-featured e-commerce platform using Django for the backend and React for the frontend. The system supports both buyer and seller roles, making it easy to manage products, orders, and payments.
## App Features

- **Buyer & Seller Profiles:**: Separate dashboards for buyers and sellers, allowing sellers to add products (with different sizes and colors, each tracking its own stock) and buyers to browse and shop easily.
- **Inventory Management:**: Sellers can view their stock levels, sales history, and see detailed reports of whatâ€™s been sold.
- **Flexible Payments**: Buyers can pay by PayPal or cash, or charge their wallet balance via PayPal for quicker future purchases.
- **Order & Transaction History**: ABuyers can view all past orders and detailed transaction records for transparency and easy tracking.
- **Smooth User Experience**: Clean React frontend that ensures responsive browsing and a streamlined shopping and management experience.


## Technologies Used

- **Django**: For building robust backend APIs and handling business logic.
- **React**: For a fast, dynamic frontend that keeps users engaged.
- **PayPal Integration**: To securely process payments and manage wallet top-ups.
- **PostgreSQL**: For reliable data storage and queries..
